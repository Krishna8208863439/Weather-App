# PowerPoint Presentation Deck (15 Slides Outline)
## Advanced AI-Powered Weather Forecasting Application

---

### Slide 1: Title Slide
- **Title**: Aura Weather AI: Advanced AI-Powered Weather Forecasting Application
- **Subtitle**: Real-time Analytics, Interactive Doppler Radar, ML Risk Predictions & Voice AI Assistant
- **Presenter**: Senior Engineering Team

---

### Slide 2: Project Vision & Motivation
- **Problem**: Legacy weather portals are cluttered, static, lack predictive risk scores, and offer zero conversational AI assistance.
- **Solution**: A modern Glassmorphic web platform combining live REST weather telemetry, interactive Doppler maps, ensemble ML predictions, and voice AI chatbot.

---

### Slide 3: Tech Stack & Tools
- **Frontend**: React 18, Vite, TypeScript, Tailwind CSS, Lucide Icons, Chart.js, Leaflet.js
- **Backend**: Python FastAPI, Uvicorn, Pydantic, ReportLab (PDF Engine)
- **AI & ML**: Scikit-Learn (RandomForest), Web Speech API (Voice Synthesis/Recognition)
- **Deployment**: Docker, Docker Compose, Nginx

---

### Slide 4: Key Features Overview
1. Real-time Weather & AQI Dashboard (12+ atmospheric parameters)
2. Interactive 48-Hour Chart.js Graphs & 10-Day Outlook Cards
3. Leaflet.js Maps with RainViewer Live Radar Tiles
4. Voice-Enabled AI Assistant (Outfits, Farming, Travel advice)
5. Scikit-Learn ML Risk Predictor (Rain %, Storm Index, Heatwave %, Flood Risk)
6. Downloadable PDF Weather Reports & Emergency Safety Hub
